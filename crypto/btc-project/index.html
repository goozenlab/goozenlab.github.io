ビットコインを使うなら、これらの重要なオープンソースプロジェクトへの支援を検討しよう 2022年1月27日 21:58

ビットコインの精神は、重要なオープンソースソフトウェアプロジェクトの範囲によって実現されています。このようなプロジェクトをサポートすることを検討してください。

ビットコイナーとしての私たちの最大の資産は、この運動を可能にするオープンソースコミュニティです。 オープンソースソフトウェアは、私たちを追跡し、検閲し、コントロールしようとし続ける政府や大企業から独立したまま、世界中の個人に力を与えています。 これらのプロジェクトの多くは、営利目的で運営されているわけではなく、大規模な予算があるわけでもありません。 その代わりに、時間とエネルギーを無償で提供する世界中の個人（フリーダム・ファイター）のネットワークに依存しているのです。

では、ビットコインをサポートする私たちは、このようなプロジェクトの必須作業をどのようにサポートできるのでしょうか。 例えば、Bitcoin 2022で、私たちの目標は、Bitcoinとオープンソースの倫理に忠実でありながら、 新規参入者を歓迎する史上最大のビットコインに焦点を当てたイベントを開くことでした。 （免責事項：Bitcoin 2022はBTC Incによって運営されており、BTCはまたBitcoin Magazineも運営しています）。

この使命を念頭に置き、オープンソースの貢献者とプロジェクトをできる限りサポートしなければならないことは明らかです。 私たちにとってこの取り組みは、オープンソース・プロジェクトとフリー・オープンソース・ソフトウェア（FOSS）の重要性だけに焦点を当てたイベントの専用会議室から始まりました。 10 月には、オープンソースの無料チケットプログラムを開始し、厳選されたプロジェクトのオープンソース貢献者に 100 万ドル相当のカンファレンスパスを提供しました。 その後、無料オープンソースチケットプログラムの対象となるプロジェクトが24件追加されました。

以下の30個のハイライトされたプロジェクトは、ビットコインの進歩のために最も重要であると私たちは考えています。読者は、それぞれの方法でこれらのプロジェクトをサポートすることを検討してください。 2021年10月1日以前にオープンソースのコードベースにコミットした人は、Bitcoin 2022の4日間のフリーパスをここに申請することができます。

  1. ビットコインコア : ビットコインネットワークのリファレンスクライアントとバックボーン - ノードとウォレットの両方。
  2. C-Lightning（シーライトニング : Lightningの主要な実装の1つで、プラグインを活用することで追加の機能や制御を行うことができます。
  3. イモータン : ライトニングネットワークのプロトコルを実装した、プライバシーに重点を置いた最小限のプロトコル。
  4. LDK : 低レベルのLightningロジックを正しく実装することを気にせずに、Lightningノードを構築できるライブラリです。LDKはRust-Lightningをベースにしており、フル機能でありながら信じられないほど柔軟なLightningの実装が可能で、使い方を自由に決めることができます。
  5. LND : 現在最も普及しているLightning Networkの実装。
  6. Tor : 監視や検閲を防ぐことに重点を置いたネットワークで、ビットコインのプロジェクトでよく利用されている。

ビットコイン・サーバー

  1. Nix Bitcoin : https://github.com/fort-nix/nix-bitcoin : セキュリティに重点を置いたフル機能のビットコインノードを簡単にインストールするためのNixパッケージとNixOSモジュールの集合体。
  2. Nodl https://www.nodl.it/ : パワフルでありながら簡単に実行できる、24時間365日稼働のビットコインノードです。
  3. RaspiBlitz : LightningとJoinMarketを簡単に利用できるビットコインフルノード実装。
  4. Braiins OS : ビットコインマイニングASICをよりコントロールし、より収益性の高い投資回収を可能にするカスタムファームウェア。

ウォレット

  1. Bitcoin Beach : コミュニティに特化したウォレットで、信頼できるコミュニティメンバーが保有する鍵によるマルチシグカストディアンに依存した簡単なインターフェースを持つ。

  2. Phenix : AndroidとiOSで利用可能な、パワフルかつユーザーフレンドリーなLightningウォレットです。

  3. Blue : AndroidとiPhoneで利用可能な、使いやすさとマルチシグなどのパワーユーザー機能を兼ね備えたパワフルなモバイルウォレット。

  4. Breez : AndroidとiPhoneで利用できる使いやすいLightningウォレットで、Podcasting 2.0とPOS機能を備えています。

  5. Muun : AndroidとiPhoneの両方に対応した非常に使いやすいモバイルウォレットで、従来のビットコイン決済とLightningの両方に対応している。

  6. マーキュリーウォレット : ステートチェーンという概念を用いて、最小限の手数料でビットコインをオフチェーンでスワップできるようにしたプライバシー重視のウォレット。

  7. Samourai : プライバシーに特化したAndroidウォレットです。チェーン上のプライバシーを向上させるために、共同取引を簡単に利用できます。Samouraiはまた、QRコードペアリングプロセスとTorで自分のノードを簡単に使用できるようにします。

  8. Zeus : AndroidとiPhoneで自分のLightningフルノードと対話する最も簡単な方法。

  9. スパローウォレット : Electrumサーバーと互換性のある、フル機能のデスクトップウォレットです。マルチシグ構成を含む主要なハードウェアウォレットに対応し、PayNym、PayJoin、Whirlpool CoinJoinのサポートを内蔵しています。

  10. スペクターウォレット : マルチシグ構成を含む、独自のノードで任意のハードウェアウォレットを簡単に使用できるデスクトップウォレットです。

  11. Electrum : ハードウェアウォレット、マルチシグ、ライトニングなど、さまざまな方法で使用できる強力なデスクトップウォレットです。

  12. Seed Signer : 50ドル程度で自作できるスクリーンとカメラ付きのハードウェアウォレット。より大きな画面とスマートカード対応を望むなら、DIYワットでSpecterを250ドル程度で作ることができる。

サービス

  1. Bisq : ビットコインをドル、ユーロ、シットコインと交換する際に、取引所に個人情報を提供することなく、他の人と直接取引できる分散型ビットコイン取引所。
  2. BTCPayサーバー : ビットコインによる決済を簡単かつ確実に行えるPOSシステム。
  3. エクレア : モバイルに特化したScalaベースのLightning実装です。
  4. Electrs : 様々なウォレットやツールで独自のビットコインノードを使用するための、簡単でパフォーマンスの高い方法です。エレクトラは、ほとんどの専用フルノード実装に組み込まれています。
  5. JoinMarket（ジョインマーケット : CoinJoinの分散型実装で、中央集権的なサーバーを介さずに共同取引を構築することにより、ユーザーにさらなるプライバシーを提供します。
  6. LNBits : 様々な場面でLightningを簡単に組み込むことができるプラットフォームです。
  7. メンプール : 自分のノードで簡単に使用できる、ビットコインネットワークの現在の状態に関する直接のビューです。
  8. RoninDojo : ElectrumとQRコードとTorによるSamouraiウォレットの簡単な統合をサポートするBitcoinフルノード実装です。
