DRAFT: project

updated at 2019-08-28

コンピューティングと書くと少し古臭い感じがしますが、どうもそれに似合う言葉がみつからなったのでそのままにします。

最初のマックは PowerBook160、マック歴もずいぶんたちました。 コンピューターこそ早くなりましたが、 そこまで劇的にコンピューターライフが変わったようには思えません。 ネットが普及して、生活の一部分に、そして全てのもの事が簡単になってきています。 ただ同時に、個人的にはどんどん窮屈な感じになってきてる、 ある意味、自由が奪われてきている感じがします。 なんか取り込まれていく感じといいますか。。

オープンソースの活動も活発であり、 少しの手間とリサーチによって、 自由度を高めることもできるようになっているので、まとめてみました。

目次

コンピュータ全般

macOSやLinuxでも使えるちょっとした事をまとめました。 フォルダーについて、バックアップについてなど、全般について、 少しだけ、すっきりします。

  1. コンピューターに名前を！：私のコンピューターの名前はガジョです。その前の奴はエビス。
  2. USBスティック自動バックアップ：rsyncとlaunchdを使ったバックアップの自動化。

Raspberry Pi

ラズベリーパイにラズビアンをインストールして使います。 同時に、オープンソース・ルーターOSのopenWrtも少し挑戦してます。

  1. Travel Router: ノマド生活で使える自前のVPN対応トラベルルーターを作ってみました。
  2. ビットコン・ノード
  3. Pihole（広告ブロッカー）

OpenWRT

  1. Archer C7
  2. Travel Router
  3. Build Own Package

Android (LineageOS+MicroG)

Gさんが裏でコソコソと何かしらあやしげな行動を取っているようなので、懸念材料を少し減らす為に なるべくオープンソースを使って、利用してます。

Hackintosh

マッキントッシュがどんどん閉鎖的になってきています。 セキュリティー上、その方が管理しやすいし、会社の信用度も他の会社に比べれば高いと思っていますが、少し残念な傾向であると思います。

弊害として、ハードウェアの自分で変更したりができなかったり、とどっぷりとアップルに依存する形であれば問題ないけれど、 いつ、G社みたいに、GoEveil化するかわかりません。

という事から、少しの手間と、少しだけ不具合があったりするようですが、ハッキントッシュというwindows機にmacOSをインストールしてしまおうと、ライセンス上も少しグレーであるようですので、参考までにとどめておいてください。

フォーラムは一杯ありますので、そちらを参考のほど。

macOS

マック使用歴もかなりになりました。あまりもう新しいものに飛びつかなくなる年頃でのあるので、

そんな中、お気に入りのソフトを紹介します。

基本あまりたくさんの、拡張機能などは入れませんし、割とデフォルトの状態で使っています。

あまり知られていない裏技も少し追加します。

  1. お勧めマックソフト：まっ、定番ばかりですが、、
  2. 自家製マックで一杯：マックを新しい世界へ、自家醸造で自分好みに。
  3. 未だに現役アップルスクリプト：アップルスクリプトで環境設定をいじる。
  4. RamDiskを起動時にマウント：キャッシュファイルをラムディスクにいれてアプリを高速化。
