DRAFT: Best Mac Software

2019-09-30-favorite-mac-softwear AFTER FINISH UP INSTALLATION.

マック歴もずいぶんたちました（最初のマックは PowerBook 160）。 色んなソフトを試してきました。 コンピューターこそ早くなりましたが、そこまで劇的にコンピューターライフが変わったようには思えません。 OSXそのものも成熟期を迎え、コンピューターの役目もどんどんタブレットに移行していっています。 とはいえども、キーボード付きのコンピューターが無くなるまではまだまだ時間がかかりそうです。

私がこよなく愛するソフトは以下。 変わり種がある感じはありませんが、記念にと言いますか、、 多分もう大きな変更があるとは思えないので公開しました。

有料ソフト：他のOSには無い優秀なアプリ

  • Affinity Designer：画像の処理はここで行ってます。GIMPもトライしたほうが良いのかも、、
  • Bear：evernoteの代替として、ローカル環境で使ってます。
  • Reeder：RSSリーダーだけでは無くて、Instapapaerも観れるのが良いです。Tiny-Tiny RSSとも問題無く動いてくれてます。
  • Hazel：地味に役に立つ自動化ソフト。デスクトップに散らばりがちなファイルを整理するのに使ったりしてます。でもちょと高いです。lanchedとスクリプトでなんとかすることもできます。
  • Tree：シンプルかつデザインの良いアウトラインソフト。リンク切れ、開発停止？
  • OmniGraffle Professional：オムニシリーズは好きだったんですけどね、

オープンソースソフト

  • nvALT
  • Visual Studio Code：テキストエディタ。他のソフトも色々試しましたがこれで落ち着く
  • Enpass：パスワードの管理には使ってます。1Passwordから移行
  • gnucash：ざっくりとお金の流れを管理してます
  • Simple Comic：漫画を読むためのアプリです
  • VLC：動画を見るならこちらが定番

ユティリティー

  • iTerm2
  • karabiner
  • Etcher：イメージをUSBメモリーなどに焼くためのソフト
  • X Lossless Decoder ：Flacファイルを変換するためのアプリ
  • dato：拡張時間表示メニューソフト

開発

  • Xcode：iOSアプリ開発のためのソフト
  • homebrew：
  • docker：
  • hugo：ブログのシステム、自分の中ではかなり理想に近いブログのファイルジェネレーター

おまけ

  • Stellarium：天体ソフトというのでしょうか、星好きの人は
  • Celestia：こちらも上と同様ですが、こちらは宇宙を駆け巡る事ができます
