De-Googled Phone

updated at 2020-12-21

携帯を持っている限り、少なからずプライバシーは侵害されてしまうのですが、 できる範囲のことはしたいと思い。 自由へむけカスタムロムを使いグーグル・アンドロイドからのオプトアウト。

De-Googled Phone と呼ばれているらしい。

現在あるアプリなどを使ってできるだけ セキュアーかつ自由な端末をつくり上げたいと思います。

システム

アンドロイドはカーネルにオープンソースのLinuxを使っています。 そして基本OSの部分は Android Open Source Project という名前で公開解されています。

セキュリティー上問題を抱えている部分は、グーグルプレイサービスところです。 そのサービスを使うことによって位置情報などの個人情報がグーグルに提供されます。

De-googled Android OS

  • GrapheneOS: https://grapheneos.org/
  • CalyxOS (microG): https://gitlab.com/CalyxOS/calyxos
  • e foundation (microG): https://e.foundation/
  • LineageOS+MicroG: https://lineage.microg.org/
  • Custom ROM + micro-g: lineageOS / AOSiPなど

※　microG: Google APIの代替アプリ

Linux Phone

  • Unbuntu Touch: 最近プロジェクトが独立 https://ubuntu-touch.io/
  • Kali hunter

アプリケーション

オープンソース

優秀な方々の奉仕により成り立っています。 愛と希望に満ち溢れた！？端末に仕上がります。 自分自身の景気が良くなったら、必ず貢献させていただきます！ アプリの質やデザインもどんどん良くなってきてます！全てお勧めのソフトです。

  • F-Droid：オープンソース・アプストア
  • Aurora：Google Store（匿名でアプリがダウンロードできますし、購入したアプリもこちらからインストールできます。）
  • Amaze：ファイルマネージャー
  • Agis：OTPアプリ、GoogleAuthentificatorの代替
  • Bitwarden：パスワード管理アプリ
  • Syncthing：ピアツーピアのファイル同期アプリ
  • TC (TrackerControll)：ファイヤーウォール＋広告ブロッカー
  • DNS66：DNSを使った広告ブロッカーとローカルDNSサーバーへ接続する
  • OandBAckup：バックアップソフト

Internet

  • Mullvad：VPNクライアント
  • K-9 Mail：Emailクライアント
  • Tutanota：Emailクライアント
  • Fennec (Firefox)：ウェブブラウザー
  • Bromite：ウェブブラウザー（要レポジトリー追加）
  • DuckDuckGO：ウェブブラウザー
  • Simplenote / S. Note：テキスト共有

Communication

  • QKSMS：SMSクライアント
  • ~Signal：メッセージングソフト~
  • Session：Signalのフォークアプリ、電話番号登録なしで使えます（要レポジトリー追加）
  • Twidere：ツイッター・クライアント
  • Tusky：マストドン・クライアント

Calender and Tasks

  • Etar：カレンダーアプリ
  • OpenTask：タスク・マネージャー、DecSync+Syncthingで端末間同期
  • DecSync：カレンダー・タスク・コンタクト同期ソフト
  • ICSx：WebCal管理

Criptos

  • Green：BTCウォレット、最近Trezorハードウェアウォレットに対応
  • Zeus：Lightningウォレット

Media

  • NewPipe：広告なユーチューブ・プレーヤー
  • AntennaPod：ポッドキャスト
  • Slide：Redditビューワー
  • Glider：HackerNewsビューワー
  • VLC：動画プレーヤー

etc

  • Gadgetbridge：Amazfit青春版（心拍数のデーター等は自分の所でキープ）
  • Suntime：日の出、日没トラッカー
  • OrganicMap：MapsMeの代替アプリ、完璧っす
  • Habits：習慣化トラッカー、デザイン使い勝手共に良いです

Cycling / Trecking

  • AAT：
  • OpenTracker：
  • BoltOn：サイクルコンピューターのWahoo Boltへの接続アプリ

クローズドソース・アプリケーション

その他オープンソースで代わりが見つからないソフトもいくつか使っています。

  • Instapaper
  • Komoot：自転車データー作成共有アプリ
  • mapsMe：旅人マストな地図ソフト（代替ソフトがないというわけではないのですが、おしゃべりなアプリです）
  • Element：自転車のガジェット管理
