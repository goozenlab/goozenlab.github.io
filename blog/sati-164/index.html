Sati v1.6.4

iOS8上での音が鳴らない不具合は

ホーム＞設定＞サティ＞通知より、

サウンドのスウィッチをオンにしてください。 これで、問題は解消するはずです。

for Bug that you don’t get any sounds on iOS8,

go Home > Setting > Sati: Notification
and turn Sound ON. This solves this problem… I’m sorry so taking looong time to fix it.

長らくご迷惑おかけしました。

そして、次のアップデートに入れる予定のものが少し入り込みます。

  • アイコンだらけになっていたメイン画面をスッキリとさせました。

  • 最近の状況を示すアイコンを中心の円の内側に入れました。

    • 今日を含めて９日分が表示されます。今日が一番上です、時計回りに過去に戻ります。
    • 瞑想回数により白が濃くなります。

  • 設定画面も少し手を入れて、

    • 静寂モード（終了時にのみ風鈴の音がなります。）を追加しました。
    • 下は終了音の選択

  • ホーム＞設定＞サティで設定できる項目を追加しました。
    • 瞑想中の音のオン・オフ
    • カレンダー書出し時のフォーマット
