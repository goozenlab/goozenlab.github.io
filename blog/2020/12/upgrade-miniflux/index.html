OpenWRT: Minifluxのアップデート

OpenWRTトラベルルーター(Raspberry Pi.3A+) で運用している、Miniflux RSSサーバーを6ヶ月ぶりにアップグレードしました。

アップグレード作業

  1. バイナリーを開発サイトよりダウンンロード
  2. バイナリーに実行権を与える
  3. サーバーのセッションを切断する
  4. Luci: ウェブ管理サイトよりサービスを止める
  5. （DBをバックアップアップとあるも無視）
  6. バイナリーを入れ替える（アップグレード）
  7. データーベースのスキームを更新（confファイルを指定する必要がある）
  8. Luci: サービスをスタート
root@OpenWrt:~# wget https://github.com/miniflux/v2/releases/download/2.0.26/miniflux-linux-arm64
root@OpenWrt:~# chmod 755 miniflux-linux-arm64-2.0.26
root@OpenWrt:~# /opt/miniflux/bin/miniflux -c /opt/miniflux/etc/miniflux.conf -flush-sessions

LUCI (System > Start Up ): Stop Process

root@OpenWrt:~# mv miniflux-linux-arm64-2.0.26 /opt/miniflux/bin/miniflux
root@OpenWrt:~# /opt/miniflux/bin/miniflux -c /opt/miniflux/etc/miniflux.conf -migrate
Current schema version: 28
Latest schema version: 42
Migrating to version: 29
......
Migrating to version: 42
root@OpenWrt:~#

LUCI (System > Start Up ): Start Process

※　Upgrading Miniflux: https://miniflux.app/docs/upgrade.html

すこし端折りましたが問題なくアップグレード完了。

以前は携帯電話のブラウザーにて記事をスワイプする時に少し引っかかりがあったりしましたが、 そういったこともなくなり、全体的に操作が快適になりました。 （GoLangのアプリは印象が良い。勉強しようかなと思ったり。。）

Pi3A+.OpenWRTトラベルルーターも良い感じで動いてくれています。
