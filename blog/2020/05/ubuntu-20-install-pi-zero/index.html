Ubuntu 20.04 on Pi Zero

先日の続きで、今度はパイゼロにUbuntuをインストールを試みました。

緑のランプが点滅している動いているのではと思っていました。 が、、どうも嫌な感じ、、前回のようにPi3Bに繋げてみたりしてもダメ。

で、ちょっと調べてみました。 若干情報が古いのですけれど、現状変わらずだと思います。

https://askubuntu.com/questions/703070/does-ubuntu-work-on-a-raspberry-pi-zero

曰く、Ubuntuのパイ・サポートはARMv7以降の端末になっていて、 パイ・ゼロのちょっと特異なARMv6には対応していないとの事。

結局のところサポート外のようです。

素直に、Raspbianを使うのがよろしいようですね。。 ちょっと他のPiと統一したかったので残念。
