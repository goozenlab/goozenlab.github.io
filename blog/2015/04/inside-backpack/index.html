バックパックの中身

ちょっと自分でも整理をしたかったので、今回の約４ヶ月間マレーシア・ネパール旅行での持ち物リストを書き出してみました。バックパックを新しくしようかなと、、 全部で多分15キロぐらい、バックパックとブリーフケースに荷物がパツパツに入ってます。 一年の旅行でもリストはほとんど同じだと思います。

ブリーフケース

MBP15が入るブリーフケース。昨年MBP15が故障。MBA11 (-1.4kg)になったというのになにか、まだずっしりとした重さがある。

  1. ポケットトートバックのような物にMBA 11と封筒ケース
  2. Kindle PaperLight
  3. リコー GR + ViewFinder
  4. iPhone 4S
  5. iPod 240GBとヘッドフォン
  6. A6 システム手帳
  7. ナカバヤシのノート
  8. マネーベルト
  9. 長財布と小銭入れ
  10. 書類入れ：A4シート、三折り書類入オレッタ

バックパック

人から譲ってもらった物ですが、サイズはパーフェクト、40L?くらい、REI社製。人に切られたり、ネズミに齧られたりでボロボロ。

服飾類

  1. ハンガー x 3
  2. 衣服メッシュケース（着ている物は除外）
    1. タオル、手拭 x 4
    2. Tシャツ x 2、シャツ x 2
    3. ズボン、パジャマ
    4. インドのドーティ(2m x 1.2mくらいの腰巻）
  3. 寝袋とショール
  4. 眼鏡ケース
  5. サンダル

生活

  1. ネティポッドとサプリメント類
  2. 洗顔キット：垢こすり、シャンプー、石けん
  3. ヘッドライト
  4. 鍵 x 3
  5. ZipLockやプラスチック数種
  6. コクヨの筆箱：生活グッズ
    1. 耳かき、爪切り、髭剃り
    2. はさみ、定規
    3. 替芯：シャープペンとボールペン

デジタル関連

  1. MacBook用電源とユニバーサルアダプター
  2. Sony GSPロガー
  3. 小型BTスピーカー
  4. eneloop： charger、AA x 5、AAA x4
  5. 外付けHD：バックアップ用として持ってくるも、壊れる。
  6. コクヨの筆箱：デジタルグッズ
    1. ケーブル類
    2. USBアダプタ
    3. USBメモリースティック

etc

  1. ユニクロのバッグ：移動時はバックインバックになっている
  2. 防水性のバッグ：バックパックのカバーが無いので、
  3. マグカップ
  4. Beefeaterのペットボトル
  5. 塩や香辛料：ほとんど使わず
  6. ジェットボイルとガス：今回特別参加
  7. 液晶・レンズクリーナー
  8. ベッド兼ヨガマット
  9. 文具： Kurutoga、消しゴム、Pilotのボールペン
  10. メッシュケース：S字フックやクリップ等が入ってます
