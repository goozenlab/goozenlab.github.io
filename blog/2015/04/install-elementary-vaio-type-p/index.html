elementaryOSをインストール

デザインがシンプルで気になっていた、elementaryOSを、開発が終了してしまったクランチバングをインストールしていた尻ポケマシーンのvaioP (新しいCPUで復活しないものか。。）に再インストールする。

以前のバージョンのLunaを入れた事があったのだが、Liveでは動くものの、インストール後のgrubが正しく設定されず、修正もできなかったので起動しなかった。

今回はオフライン気味な端末として、物書き、お小遣い帳などとして活用する。そしてファイルの共有は下記のお勧めソフトのSyncthingを使う。なるべくサードパーティーのサービスは使わない。（勝手なマイポリシー）

事前準備

  1. isoファイルのダウンロード。
  2. USB起動ディスクの作成ソフトのダウンロード
  3. USB起動ディスクの作成

インストール

USBメモリースティックを入れて起動。後はそのまま手順にそっていけば問題ない。パーティションの所だけ、各自の設定により変える必要がある。（本機はLinuxオンリーマシーンなのですべてをデフォルトの構成）

追加ソフト

長い事Linuxを使っている訳では有りませんが、お勧めのソフトは以下、

  1. Syncthing: P2P同期ソフト（BittorrentSyncのオープンソース判）
  2. Mozc: 日本語の変換ソフト
  3. Editor: 集中して物を書くときのソフト
