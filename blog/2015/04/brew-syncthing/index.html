syncthingをマックにインストール

Dropboxの容量が中途半端だったり、ちょっとうまく動かなかったりでオープンソースで代替の物を探していたら、ownCloudとSyncthingというソフトを見つける。

ownCloudは速度に問題があるようなので、ピアツーピアでの同期オープンソースアプリ、Syncthingをマックにインストールする。

Syncthing: open source bittorrentSync

バイナリーがサイトよりダウンロードする事がでるが、brewより一発でインストール。

常時稼働のサーバーか同時に動いている別の端末が必要なので、Debian-VaioPを取りあえず使う。SynologyのNASは実家にほったらかしになっているので後々追加予定。

$ brew install syncthing
==> Downloading https://homebrew.bintray.com/bottles/syncthing-0.10.30.mavericks
######################################################################## 100.0%
==> Pouring syncthing-0.10.30.mavericks.bottle.tar.gz
==> Caveats
To have launchd start syncthing at login:
    ln -sfv /usr/local/opt/syncthing/*.plist ~/Library/LaunchAgents
Then to load syncthing now:
    launchctl load ~/Library/LaunchAgents/homebrew.mxcl.syncthing.plist
==> Summary
🍺  /usr/local/Cellar/syncthing/0.10.30: 6 files,  10M
$

起動時にスタートするには上にある通り、LaunchAgentsに登録します。

$ ln -sfv /usr/local/opt/syncthing/*.plist ~/Library/LaunchAgents
$ launchctl load ~/Library/LaunchAgents/homebrew.mxcl.syncthing.plist
