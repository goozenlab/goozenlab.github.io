profile-sync-daemon

https://wiki.archlinux.jp/index.php/Profile-sync-daemon

jin@debian:~$ sudo apt install profile-sync-daemon

jin@debian:~$ psd First time running psd so please edit /home/jin-debian/.config/psd/psd.conf to your liking and run again. jin@debian:~$

jin@debian:~$ vi .config/psd/psd.conf

USE_OVERLAYFS=“yes” BROWSERS=“firefox”

jin@debian:~$ sudo modprobe overlay

/etc/sudoers に以下の行を設定することでユーザーに権限が与えられます。root で /usr/bin/visudo を実行して追加してください:

jin ALL=(ALL) NOPASSWD: /usr/bin/psd-overlay-helper

jin@debian:~$ psd p I require modinfo but it’s not installed. Aborting!

/sbinへのパスが通ってないのが問題のようなので、 .profileにPATHを追加する

PATH=“/usr/sbin:/sbin”

で、

jin@debian:~$ psd p Profile-sync-daemon v6.31 on Debian GNU/Linux 10 (buster)

Systemd service is currently inactive. Systemd resync-timer is currently inactive. Overlayfs v23 is currently active.

Psd will manage the following per /home/jin-debian/.config/psd/psd.conf:

browser/psname: firefox/firefox owner/group id: jin/1000 sync target: /home/jin-debian/.mozilla/firefox/25pb77js.default tmpfs dir: /run/user/1000/jin-firefox-25pb77js.default profile size: 4.0K overlayfs size:
recovery dirs: none

browser/psname: firefox/firefox owner/group id: jin/1000 sync target: /home/jin-debian/.mozilla/firefox/6uxssp9i.default-esr tmpfs dir: /run/user/1000/jin-firefox-6uxssp9i.default-esr profile size: 51M overlayfs size:
recovery dirs: none

jin@debian:~$

ユーザーモードで立ち上げます。
