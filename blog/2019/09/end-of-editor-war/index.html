エディター紛争

彼ノ地タイ国ニテ終止符ヲ打ツ。

端末の設定やら整理やら、ちょっと環境を再構築中です。

ここ最近の傾向でなるべく・オープンソースかつマルチプラットフォームのものを選ぶようにしてます。 で、いろいろ使い回してたエディターもようやくひとつに絞りました。

Visual Studio Code

Visual Studio Code（通称：VSCode）は2015年11月18日にMicrosoftよりリリースされた、 オープンソースのソースコードエディタです。

マックユーザーとしては不覚ながら、マイクロソフト製品。。（でも最近は柔軟化してるというか、、サティア氏の影響なのでしょうか。）

しかし、自分ルールの範囲内であるし、 マイクロソフトのソフトはデザイン的に良い印象が無いのですが、 画面の下に表示される色のついた帯のデザインが割とお気に入りでもある。

他の候補は

  • Sublime Text：無料で使えるが、有料ソフト
  • Atom：起動が遅かったりして、エディターに求める軽快さがない
  • TextMate：スクリプトを書いたりするには少し機能が足りず
  • vim/emacs：あぁ〜〜、ちと無理。

ただし、未だ完全にはtextmateをアンインストールはしてません。 ちゃちゃっとスクリプトやテキストファイルを修正するにはこっちのほうが軽快なので、 もう少ししばらくキープ。。 （というか、強引にアンインストールしたほうがいいかもしれんのかも。。）

そして、ブログ書きにはnvALTが未だ定位置を譲らず活躍。

慣れるまで少し時間がかかるかもしれないけれど、もう浮気は面倒なのでしっかりと使っていきたいなと思ってます。

そして、iPhone11が発表される。。iOS13がすごく気になるが、 最後の噂を信じ来年春までiPhone5Sでもう少し引っ張る。。 欲しいんだけど、たっかっすっぎーー。

で、欲望を分散するため？ パイ４をオーダーしちゃいました。。なんにつかお。。
