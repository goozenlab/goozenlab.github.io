youtube動画をダウンロード

ユーチューブで気になった動画があったので、ダウンロードをする方法を探しました。 割とあっさり見つかり、割と簡単。。

インストール

さて、インストールはbrewで一発。

guiness:~ jin$ brew install youtube-dl

使い方

使い方はシンプル。動画のリンクをコピペで行けます。

インストールをするにあたって まずはダウンロードのフォーマットを決めます。 コマンドは youtube-dl -F <<ADDRESS>> でダウンドード可能な様々なサイズ・フォーマットが表示されます。

ダウンロードは youtube-dl -f #FORMAT <<ADDRESS>> オプションは小文字のfです。FORMATはリストの一番最初にある数字を入れます。
