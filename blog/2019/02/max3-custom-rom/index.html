Xiaomi Max3のカスタムロム

７インチ近い液晶画面を持つMi Max3、現在MIUIeu Betaを使ってます。MiUIもこなれてきていて、使い勝手も悪くはありませんが、LineageOS16の移植を待っています。

最近の端末は、Project Trebleに対応しているものが多く、Mi Max3もそのうちの一つ。

Device Table
Nokia 7 Plus b2n A/B
Xiaomi Mi Max 3 nitrogen A

このプロジェクトの利点は、いろいろあるみたいですが、 端末に限定されたカスタムロムの必要が無くなるらしい、、 ということで簡単に他のカスタムロムも動くはず、、 様々なカスタムロムが走るのかと思いましたが、そうでもない様です。 試した人がいないということだけかもしれませんが、あまり活発な動きが見当たりません。

そこで、すこしカスタムロム関連の動向をチェック。以下のカスタムロムが見つかりましたので、 ちょっとリストにしてみます。（XDAのフォーラムより）

Custom Rom Base Install
ResurrectionRemix (RR 7.0) Oreo GSI Install guide
Pixel Experience Pie GSI Install guide
AOSP Treble Pie GSI Install guide
AICP-M Treble Oreo GSI Install info
HAVOC Pie GSI Install guide

※ GSI: Generic System Image

まぁ、あるといえばある。。

そして、リカバリーツールのTWRP も安定したものが出ました。 パーティションが暗号化されていたりと問題がいろいろとあったようですが、これで、フルバックアップ・復元が可能になります。新しいロムを試したりが安心して簡単にできる、、とのこと（未確認）。

ただ、私のお目当は、LineageOS16。

MiMaxでLineageOSを移植していた開発者・nijel8さんが開発宣言をされていて、 じわじわと、Lineage16のカスタムロムの移行が行われています。。。

、、と思います。

非常に楽しみにしています。
