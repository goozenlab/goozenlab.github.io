INBOXを使い続ける

さて、日々常用している、携帯ソフトの中で多分一番使っている頻度の高いアプリ、INBOXが利用停止になりました。GMAILへの移行を強制されます。

試しにGMAILを使ってみるも、あまりピンとこない。GMAILの進化を期待していただけに、非常に残念。慣れの問題なのかもしれないけれど、INBOXの方が未だ使いやすいと思う。

さて、グーグルはこのアプリの会社を買い取って一体何がしたかったのだろうか？ 結局、競合会社を無力化するためだけの買収だったのかと、、憤りすら感じる。

最近、ちょっとだけ若い人と話した時には、「もうメールはあまり使わないし、久しぶりにメールで写真を送りましたよ」、なんて言われたので、もうあまりメールアプリは重要ではないということなんだろうか？

、、せめて、放っておいてもらっても良いのだけどね、、

で、ありがたいことに、xdaにて使用不可を回避する Tip がのっていました。Root化が必要です。

https://www.xda-developers.com/use-inbox-by-gmail-android-after-shutdown/

INBOXを起動した状態で、ターミナルソフト立ち上げ、

# su
# am start -n "com.google.android.apps.inbox/com.google.android.apps.bigtop.activities.SwitchActivity" --ez "countDown" true

を実行し、もう一度、INBOXを起動して、継続使用可能になります。もう一つの端末、Nokia 7+はルート化されていないので、ここ最近、iOSより流れてきた、Sparkをインストールしました。

はぁ、（すっきりしない、ため息）
