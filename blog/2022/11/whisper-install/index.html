Whisperをインストール

OpenAI Whisper Transcription Testing

OpenAI Whisper Transcription Testing https://blog.lopp.net/openai-whisper-transcription-testing/

の記事（ビットコインナーさんのフォロワー）をみて結構使えるとのことで、 ブログであげているYutubeの文字起こしに使えそうなので、インストールして試してみました。

jin@bohemia ~ % python3 -V Python 3.8.9

$ brew install python

$

brew install virtualenv

virtualenv env/ source env/bin/activate

*FYI: You need to run the command to use Whisper whenever you want. This is the activation command.

Rust

Rustを使ったプログラミングを行えるように、Rustコンパイラのアップデートや開発に必要なツール類を管理するための rustup をインストールします。

brew install rustup-init

Whisper install

Requirement

$ brew install rust
$ brew install ffmpeg

依存するソフトウェアも割とたくさんインストールされます。 rustはエラーが出たらインストールを。（問題が出るとか出ないとか、私は既にインストール済みだったので検証できず）

そして最後にメインのWhisperをインストールします。 pipのバージョンが古いとインストール後に注意fが出たので、事前にアップデートしておくのが良いです。

% python3 -m pip install --upgrade pip
% pip3 install git+https://github.com/openai/whisper.git

アップグレードは以下のコマンドより行います。

$ pip3 install --upgrade --no-deps --force-reinstall git+https://github.com/openai/whisper.git

パケージ・マネージャ、pipかBrewか悩みましたが、Brewに全てのパッケージがある様ではないので、pipで統一しときました。 Brew vs. Pip: Which Package Installer Should You Use? https://www.stickyminds.com/article/brew-vs-pip-which-package-installer-should-you-use

日本語の文字起こしは言語をしてして、

whisper japanese.wav –language Japanese

Adding –task translate will translate the speech into English:

英語に翻訳したいときは、--task translateを追加します。逆方向もできるのかな。。

whisper japanese.wav –language Japanese –task translate
