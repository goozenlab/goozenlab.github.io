家鯖

友人宅で運用していたサーバー機は、現在実家の電話台の下に置かせてもらっています。 熱を持つキーボード部をなるべく大気に触れられるようにノートブックが不思議な体勢になっています。 SSDという事もあって問題は無い事と。。。

お茶菓子を入れる小さな木製のお皿の上に本体をのせ、さらにモニター部には箸置きを二つ。 その上に無線ルーター機が置いてあります。 近くには蚊取り線香のベープアース。 横には100均で買った、電源とケーブルを入れたピンクメッシュ書類ケース。 その横には電話帳となぜか子鹿物語の本。

生活感があふれています。 ホームアプライアンス感が滲んでます。

手を指し入れるとほのかに暖かみを感じる事ができます。コンピューターの温度は50度後半で推移。 23区内ですからこの夏の計画停電実施の可能性は低いと思いますが、どうにか夏を落ちずに乗り切ってもらいたい。。。

で、私は節電！節約？のためカナダに旅立ちます。 バンフという町に逗留予定です。山に引きこもりにいってきます。 厄年、天災続きのここ最近ですが、 くれぐれもグリズリーベアの餌にならないように気をつけたいと思います。
