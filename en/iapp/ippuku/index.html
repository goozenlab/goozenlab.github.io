ippuku

This is a simple software to log your activities.

How to Use

◇ Timer: start with + button on navigation bar
◇ on Cell
・Left tap: change date background color
・Right tap: change bar color on/off
・Center Double tap: delete or edit event

◇ on Navigation Bar
・Double tap: change List page (all, warm, cold, and soft color) ・Long tap: change view style (Default, 22:00 - 8:00, 5:00 - 19:30, 13:00 - 23:00)

※ There is indication on title with dot: (•title: am, title•: pm)

◇ App Setting

・You can change title of navigation bar.
